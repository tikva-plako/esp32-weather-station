# ESP32 Weather Station

Reads temperature and humidity with DHT22 sensor
and displays data on a web page via WiFi.

## Hardware
- ESP32 DevKit
- DHT22 sensor
- Breadboard + jumper wires

## Setup
Replace in sketch.ino:
- YOUR_WIFI_SSID
- YOUR_WIFI_PASSWORD

## Built with
- Arduino IDE
- Wokwi simulator
